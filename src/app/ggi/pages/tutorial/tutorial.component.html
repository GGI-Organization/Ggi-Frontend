Antes de poder realizar cualquier comando para iniciar tu proyecto debe tener instalado Node.Js validando colocando en su aplicacion de comando

node -v

Luego de tenerlo instalado puede seguir los siguientes pasos dependiendo que tipo de tecnologia esta desarrollada su aplicación.


¿Como levantar un proyecto en Angular?

1. Preparando el ambiente

Primero instalemos Angular-Cli, ejecutamos:

npm install -g @angular/cli

Esto instalará globalmente la herramienta angular-cli. Si todo salio bien, el comando “ng -v” nos mostrara las versiones instaladas:

ng -v

2. Abrir el proyecto creado

Vamos al directorio en el que creamos el proyecto:

cd nombre-del-proyecto

3. Ejecutar la aplicación

Para ver la aplicaicon en el navegador ejecutamos:

ng serve --open

Usando la opción — open (o simplemente -o) nos abrirá automáticamente la aplicación en el navegador en “http: // localhost: 4200 /”

¿Como levantar un proyecto en React?

1. Abrir el proyecto creado

Vamos al directorio en el que creamos el proyecto:

cd nombre-del-proyecto

2. Ejecutar la aplicación

Para ver la aplicaicon en el navegador ejecutamos:

npm start

Se levantar la pagina web en tu navegador predeterminado

¿Como iniciar rapido un proyecto en VueJs?

1. Preparando el ambiente

Primero instalemos Vue-Cli, ejecutamos:

npm install -g @vue/cli

Esto instalará globalmente la herramienta angular-cli. Si todo salio bien, el comando “vue -v” nos mostrara las versiones instaladas:

vu -v

2. Abrir el proyecto creado

Vamos al directorio en el que creamos el proyecto:

cd nombre-del-proyecto

3. Ejecutar la aplicación

Para ver la aplicaicon en el navegador ejecutamos:

ng run serve

Una vez hemos hecho esto, tendremos arrancado un servidor de desarrollo en nuestro equipo para ver el proyecto en un navegador a medida que modificamos el código.
